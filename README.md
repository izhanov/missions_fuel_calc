# missions_fuel_calc

# Dependencies

- Ruby `2.7.0`
- RSpec `3.9.0`

# Usage

1. After cloning repo go to this directory:
`cd missions_fuel_calc/`

2. Start the console:
`bin/console`

3. Run the code:
`FuelCalculator.call(28801, [[:launch, 9.807], [:land, 1.62], [:launch, 1.62], [:land, 9.807]])`
`=> 51951`
